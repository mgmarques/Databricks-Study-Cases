# GenAI
GenAI is built on advanced machine learning models (especially foundation models) 
that learn patterns from massive datasets and then generate original outputs.

Examples:
* Text → essays, emails, chat responses
* Images → artwork, designs
* Code → full programs
* Audio → music, voice

Well-known tools include:
* ChatGPT
* DALL·E
* Midjourney
* GitHub Copilot

## Key Concepts Behind GenAI
**1. Large Language Models (LLMs)**
* Power tools like ChatGPT
* Predict and generate human-like text

**2. Transformers**
* Architecture that enables understanding context and relationships in data

**3. Prompting**
* The way you ask the AI determines the output quality

**4. Fine-tuning & RAG**
* Customizing models with company data
* Retrieval-Augmented Generation (RAG) = combining AI + databases

## GenAI vs Traditional AI

| Traditional AI           | GenAI                     |
| ------------------------ | ------------------------- |
| Classifies data          | Creates new content       |
| Predicts outcomes        | Generates outputs         |
| Example: fraud detection | Example: writing a report |
